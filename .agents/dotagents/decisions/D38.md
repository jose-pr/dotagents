# D38: host-native provider routing

Date: 2026-07-16

## Decision

The default provider lane is chosen by the agent that calls the plan executor:
Codex uses OpenAI, Claude uses Anthropic, and Gemini uses Google. The role table's
other provider columns are explicit fallbacks only when the caller permits a
cross-provider run.

## Rationale

The same role plan should behave naturally on each agent surface and should not
silently jump providers because a table happens to list OpenAI first. Subscription
access and actual callability remain separate checks.

## Changes

- `overlays/flows/MODELS.md` documents host-native selection and fallback policy.
- `overlays/flows/flows/EXEC.md` makes the host/provider resolution step explicit.
- Installed `~/.agents` mirrors the same routing rule for the current machine.
